About Chanchu

Medical Prescription OCR System

Project Overview

Chanchu is an advanced Optical Character Recognition (OCR) system specifically designed to read and interpret handwritten medical prescriptions. This project was developed as a college project to demonstrate the application of computer vision, machine learning, and natural language processing in the healthcare domain.

The system uses a combination of image preprocessing techniques, deep learning models, and medical terminology recognition to accurately extract text from prescription images, identify medications, and provide structured output that can be easily interpreted.

Key Features:
  • Advanced Image Preprocessing: Applies multiple filters and transformations to enhance handwriting visibility and improve OCR accuracy.
  • Multiple OCR Engines: Combines the power of various OCR technologies including Tesseract OCR and custom CNN models.
  • Medical Dictionary Integration: Incorporates a comprehensive database of medication names, dosages, and common abbreviations.
  • High Accuracy: Achieves over 90% accuracy in text recognition and medication identification.
  • User-Friendly Interface: Provides an intuitive web interface for uploading prescriptions and viewing results.
Technical Architecture
Technology Stack:
  • Backend: Python, Flask
  • Computer Vision: OpenCV, PIL/Pillow
  • OCR Engines: Tesseract OCR, Custom CNN models
  • Machine Learning: PyTorch, CNN architectures
  • Frontend: HTML, CSS, JavaScript, Bootstrap
System Workflow:
  1. Prescription image is uploaded through the web interface
  2. Image preprocessing enhances quality and readability
  3. OCR engine extracts raw text from the processed image
  4. Text preprocessing cleans and normalizes the extracted text
  5. Medical dictionary matching identifies medications and dosages
  6. Results are displayed in a structured, user-friendly format
